# Roadmap

This roadmap tracks major outcomes only. Task-level tracking belongs in `docs/05-Execution-Board.md`.

## Now (Priority Focus)

1. Navigation and curated collections reliability at scale.
2. Admin authoring scalability (child linking and discovery UX).
3. Media/card tagging model alignment (parallel, non-merged behavior).
4. Technical hardening: lint, QA, deployment readiness.

## Next

1. Gallery experience consistency (view/page mosaic and admin mosaic parity).
2. Card admin UX polish (excerpt controls, search ergonomics, edit exit behavior).
3. Post-import aggregation workflows in media admin.

## Later

1. Performance/platform optimization passes (hydration tiers, deduping, provider cleanup).
2. Optional index/search infrastructure if Firestore browse/search limits emerge.
3. Extended people relationship modeling and advanced recognition workflows.

## Parked

1. Multi-author archival model.
2. Broad social interaction mechanics.
3. Full video parity across all card surfaces.
